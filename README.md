# Анализ данных о погоде в Австралии для прогнозирования осадков

Этот репозиторий содержит исследование набора данных о погоде в Австралии с целью выявления ключевых закономерностей в метеорологических показателях и разработки модели для прогнозирования осадков с использованием инструментов бизнес-анализа. В проекте применяются методы ETL-процессов, анализа данных и визуализации в Яндекс DataLens.

## Цель работы

Целью курсового проекта является исследование предоставленного набора данных о погоде в Австралии, выявление ключевых закономерностей в метеорологических показателях и разработка модели для прогнозирования осадков с использованием инструментов бизнес-анализа, включая ETL-процесс и визуализацию в Яндекс DataLens.

## Задачи

### 1. Подключение к базе данных и ETL-процесс

- Подключиться к базе данных SQLite, содержащей данные о погоде.
- Провести ETL-процесс (Extract, Transform, Load):
  
  - **Extract**: Извлечь данные из таблиц базы данных.
  - **Transform**: Очистить данные (удалить дубликаты, обработать пропуски), преобразовать данные для анализа.
  - **Load**: Агрегировать данные для формирования аналитических датасетов и подготовить данные для дальнейшего анализа и визуализации в Яндекс DataLens.

### 2. Анализ данных

Ответьте на следующие вопросы:

1. Как распределяются минимальная и максимальная температура по местоположениям?
2. Какие местоположения имеют наибольшее количество дней с осадками?
3. Как влажность и давление влияют на вероятность осадков?
4. Как направление и скорость ветра коррелируют с осадками?
5. Какие местоположения имеют наибольшую разницу в давлении между утром и днем?
6. Как облачность влияет на температуру и осадки?

### 3. Визуализация данных в Яндекс DataLens

Используйте функционал Яндекс DataLens для создания визуализаций, подтверждающих ваши выводы. Интерпретируйте их для более глубокого анализа.
